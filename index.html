Belum punya API key? Gratis, ikuti ini ↓
  1. Buka aistudio.google.com lalu login Google.
  2. Klik “Get API Key” → “Create API key”.
  3. Salin key, lalu batasi (restriction) ke domain GitHub Pages-mu sebagai HTTP referrer (mis. https://namamu.github.io/*) — wajib agar key aman & tidak ditolak.
  4. Tempel key di kolom atas. Key disimpan hanya di browser ini, tidak dikirim ke mana pun selain Google.
