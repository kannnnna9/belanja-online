Belum punya API key? Gratis, ikuti ini ↓
  1. Buka aistudio.google.com, login Google.
  2. Klik “Get API Key” lalu “Create API key”.
  3. Salin key-nya, tempel di kolom atas. Key cuma disimpan di browser ini, tidak dikirim ke mana-mana selain Google.
  4. Key baru (diawali AQ.) sudah aman otomatis, langsung jalan. Kalau key-mu yang lama (AIza…), batasi dulu di Google Cloud Console sebagai HTTP referrer ke domain yang ada di address bar (mis. https://kannnnna9.github.io/*).
