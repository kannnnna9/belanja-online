Scan label harga di rak pakai kamera, harga & nama dibaca otomatis oleh AI.

Belum punya API key? Gratis, ikuti ini ↓
  1. Buka aistudio.google.com, login dengan akun Google kamu.
  2. Buka sidebar di sebelah kiri, klik ikon kunci di bagian bawah.
  3. Di halaman baru, klik “Create API key”, pilih project atau bikin baru.
  4. Salin key-nya (yang baru diawali AQ.), tempel di kolom atas. Key cuma disimpan di browser ini, tidak dikirim ke mana-mana selain Google.
  5. Key AQ. sudah aman otomatis (terbatas ke Generative Language API), langsung jalan tanpa atur apa-apa.
